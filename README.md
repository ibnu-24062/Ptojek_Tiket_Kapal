# PROJECT-UAS-PBO-KELOMPOKk-3
Sistem Penjualan Tiket Kapal adalah aplikasi berbasis Python (CLI) yang digunakan untuk melakukan pemesanan tiket kapal laut berdasarkan rute perjalanan, kelas tiket, dan jumlah tiket. Aplikasi ini menghitung total biaya secara otomatis dan menampilkan struk pembelian kepada pengguna.
